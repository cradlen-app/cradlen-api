# Patient Org Enrollment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Introduce a `PatientOrgEnrollment` record that gates when a patient appears in an org's roster — `PENDING` at booking, `ACTIVE` on first check-in — and auto-expires bookings that were never attended.

**Architecture:** A new `patient_org_enrollments` table with a partial unique index guards per-(patient, org) membership. `bookVisit` creates `PENDING` enrollment; `updateStatus(CHECKED_IN)` activates it; the cascade cleanup soft-deletes it (and orphaned patients) when all visits are cancelled/no-show. A nightly scheduled job marks expired `SCHEDULED` visits `NO_SHOW`, triggering the existing cascade. The `findAll` patients query is gated through the enrollment table.

**Tech Stack:** NestJS 11, Prisma 7, PostgreSQL (Neon), `@nestjs/schedule` (already in package.json), Jest

```typescript
import { Injectable } from '@nestjs/common';
import { Cron, CronExpression } from '@nestjs/schedule';
import { PrismaService } from '@infrastructure/database/prisma.service';

@Injectable()
export class PatientEnrollmentCleanupService {
  constructor(private readonly prismaService: PrismaService) {}

  @Cron(CronExpression.EVERY_DAY_AT_2AM)
  async sweepExpiredVisits(): Promise<void> {
    const now = new Date();
    const overdueVisits = await this.prismaService.db.visit.findMany({
      where: {
        status: 'SCHEDULED',
        scheduled_at: { lt: now },
        checked_in_at: null,
        is_deleted: false,
      },
      select: {
        id: true,
        episode: {
          select: {
            journey_id: true,
            journey: {
              select: { patient_id: true, organization_id: true },
            },
          },
        },
      },
    });

    for (const visit of overdueVisits) {
      const journeyId = visit.episode.journey_id;
      const patientId = visit.episode.journey.patient_id;
      const organizationId = visit.episode.journey.organization_id;

      await this.prismaService.db.$transaction(async (tx) => {
        await tx.visit.update({
          where: { id: visit.id },
          data: { status: 'NO_SHOW' },
        });

        const [realCount, liveCount] = await Promise.all([
          tx.visit.count({
            where: {
              episode: { journey_id: journeyId },
              checked_in_at: { not: null },
              is_deleted: false,
            },
          }),
          tx.visit.count({
            where: {
              episode: { journey_id: journeyId },
              is_deleted: false,
              status: { notIn: ['CANCELLED', 'NO_SHOW'] },
            },
          }),
        ]);

        if (realCount === 0 && liveCount === 0) {
          await tx.visitEncounter.updateMany({
            where: { visit: { episode: { journey_id: journeyId } }, is_deleted: false },
            data: { is_deleted: true, deleted_at: now },
          });
```
